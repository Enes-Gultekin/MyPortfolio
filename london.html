Spatial Analysis of Fast-Food Venues in London

Using Foursquare venue data to test clustering vs randomness (CSR / G-function) and spatial autocorrelation (Moran’s I / LISA) across London wards.

Study Area

Greater London (33 boroughs, 649 wards)

Tools

R (sp, sf, spatstat, spdep), Foursquare API, QGIS/ArcGIS (optional)

Data

Foursquare venues (Fast Food category) + London borough/ward shapefiles + ward population (2023)

Project Overview

This project investigates how major fast-food chains are distributed across London. Using Foursquare “Dining & Drinking” venue data, we extracted fast-food venues and focused on three brands: McDonald’s, Burger King, and Five Guys. The main goal was to determine whether venue locations are randomly distributed or clustered, and whether clusters show spatial autocorrelation across wards.

Objectives

  • Collect and clean fast-food venue locations using the Foursquare API.
  • Visualize spatial concentration using maps and heatmaps.
  • Test Complete Spatial Randomness (CSR) using quadrat tests and G-functions.
  • Evaluate ward-level spatial autocorrelation using Moran’s I and LISA.
  • Check whether venue counts relate to ward population size.

Methodology

Venue data were retrieved through the Foursquare API, cleaned for duplicates and missing values, and converted into a SpatialPointsDataFrame in WGS84 for mapping and spatial operations. Borough and ward boundaries were imported from shapefiles.

Spatial Point Pattern Analysis (CSR + G-function)

We tested whether points follow Complete Spatial Randomness (CSR). CSR was evaluated using a quadrat chi-squared test and supported with the G-function (nearest-neighbor distribution) to inspect clustering across distances.

Test Metric Result Interpretation
CSR (Quadrat test) Chi-squared / p-value χ² = 1045.7; p < 2.2e−16 Strong evidence of clustering (not random)
G-function Empirical vs envelope Supports clustering overall Clustering becomes clearer at larger distances

Population vs Venue Counts

We examined whether wards with larger populations have more fast-food venues. The relationship is weak: the correlation coefficient is r = 0.242, indicating only a minor positive association and no strong evidence of dependency.


Spatial Autocorrelation (Moran’s I + LISA)

Ward-level venue counts were tested for spatial autocorrelation. The global Moran’s I test was not significant (p = 0.4489), and Moran’s I was extremely small (I = 0.0014), suggesting weak global spatial autocorrelation.

However, local cluster detection with LISA identified 36 significant wards (p < 0.05): 4 high–high clusters and 32 low–low clusters. This indicates that while global autocorrelation is weak, there are localized pockets of clustering.

Figure 6: LISA cluster map (High–High and Low–Low).

Results

  • Fast-food venues show strong clustering (CSR test significant; G-function supports clustering).
  • McDonald’s dominates the dataset (104/131 venues).
  • Population explains venue distribution poorly (r = 0.242).
  • Global spatial autocorrelation is weak (Moran’s I not significant), but local clusters exist (36 wards significant in LISA).

Conclusion

The analysis shows that fast-food venues in London are not randomly distributed; they form clusters. Yet these clusters do not translate into strong ward-level spatial autocorrelation globally. Instead, clustering is concentrated in a limited set of local hotspots and coldspots. Overall, the project demonstrates how combining point pattern statistics (CSR/G-function) with spatial autocorrelation tools (Moran’s I/LISA) provides a nuanced understanding of urban venue distribution. :contentReference[oaicite:7]{index=7}

Key Insights

CSR rejected: venues cluster strongly (χ² = 1045.7; p < 2.2e−16).

Local hotspots exist: 36 significant LISA wards (4 high–high, 32 low–low).

Population is not enough: weak relationship with venue count (r = 0.242).

Best takeaway: point clustering ≠ strong global autocorrelation across wards.
